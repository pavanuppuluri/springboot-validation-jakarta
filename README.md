# springboot-validation-jakarta
Request validation using jakarta validation api

### API Request Example

- **URL**: `http://localhost:8080/api/users`
- **Method**: `POST`

**Request Body:**
```json
{
    "name": "t",
    "email": "email",
    "password": "p"
}
```
